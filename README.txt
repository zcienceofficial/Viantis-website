DEPLOYMENT
----------
1. Back up your current GitHub repository.
2. Extract this ZIP.
3. In GitHub, upload ALL extracted files and folders to the repository root.
4. Replace existing files when GitHub asks.
5. Commit changes.
6. Wait 1-3 minutes for Cloudflare Pages to deploy.
7. Test:
   https://www.viantislifesciences.com/
   https://www.viantislifesciences.com/viacal.html
   https://www.viantislifesciences.com/privacy.html
   https://www.viantislifesciences.com/sitemap.xml
8. In Google Search Console, inspect the Via-Cal URL and request indexing.
